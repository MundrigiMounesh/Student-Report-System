# 🎓 Student Report Management System

A simple Python console application that collects student information, accepts marks for five subjects, performs calculations, and displays a formatted student report.

## 🚀 Features

- Collect student details
- Enter marks for 5 subjects
- Calculate:
  - Total Marks
  - Average Marks
  - Percentage
  - Highest Marks
  - Lowest Marks
- Display a formatted student report

## 🛠️ Technologies Used

- Python 3.x
